# CIST 2026 Conference Website

Conference on Information Systems and Technology 2026
**October 31 – November 1, 2026 · San Francisco, CA**

## Local Development

```bash
# Install Hugo: https://gohugo.io/installation/
hugo server -D
# Open http://localhost:1313
```

## Deployment

Pushes to `main` auto-deploy to GitHub Pages via the included workflow.

## Editing Content

All content lives in `content/` as Markdown files. Edit dates, text, and details there. The site design is in `layouts/` and `static/css/style.css`.
